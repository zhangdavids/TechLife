函数是第一等公民

函数不仅可以封装代码、分割功能、解耦逻辑

还可以化身普通的值 在其他函数之间传递、赋予变量、做类型判断和转换等等

就像切片和字典的值一样

函数值可以由此成为能够被随意传播的独立逻辑组件（或者说功能模块）
